系所簡介

壹、系所簡介

由於資訊科技的日新月異，資訊人才的需求日益增多，本系（含碩士班）於2007年8月1日正式成立，屬非師資培育學系，成立當天由劉校長源俊、黃院長萬居及賴主任阿福共同揭牌，且在眾多賓客之祝賀下舉行揭幕茶會。為提供資訊產業人員進修管道，本系於2011年成立資訊科學系在職碩士專班。為配合學校中長程校務發展計畫，本系於2014年將數位學習碩士學位學程納入本系碩士班之é數位學習組û。本系師資目前共十一人，其中教授二人、副教授六人、助理教授三人。教師專長均為資訊或電機相關領域專長之博士，完全符合本系的定位與系所培育目標，在教學、研究與服務上均有不錯表現。

一、本系課程注重資訊理論與實務技術，以培育計算機軟體、硬體系統之理論、開發、應用及研究之資訊科學人才。
二、結合本校教育專業與人才培育之優勢及資訊技術，培養資訊教育和數位學習科技應用及研究人才。
三、本系定位為「教學為主、研究為先之優質系所」，培育目標為「培育軟體開發及程式設計、網路與多媒體應用、數位學習的資訊人才」。
四、本系特色為養成紮實的資訊基礎能力，一切從基本功做起，以因應資訊時代的快速變遷。

貳、本系教育目標

一、培養資訊科學知識與技能，發揮創新研發能力。
二、兼具理論與實務的專業知能。
三、建立終身學習能力，掌握科技變化趨勢。
四、連結資訊科技與人文藝術之素養。

參、本系基本素養

一、具備學術倫理、資訊倫理。
二、具備人文素養及專業道德責任的人文關懷。
三、具備外語溝通與國際觀。
四、具有共通基礎、旁通廣博及貫通融合的三通素養。
五、具有邏輯思考的科普素養。

肆、本系核心能力

一、程式設計能力。
二、計算機系統基礎能力。
三、數位科技應用能力。
四、問題解決及趨勢掌握能力。
五、團隊合作與創新能力。
